# Cash On Click

## Project Description

App for Customers to Order Cash directly to their home and cash points to check for orders in their area and mark delivery
Also provide live tracking for any order and will have backend reconcialiation reports

We have also built an admin dashboard using Metabase (An open source BI tool) for live monitoring of area operations

(We plan to add a manifest file to make into a PWA, so that it becomes platform independent)

#### Installation Steps

* Install Frappe (https://github.com/frappe/bench.git)
* bench get-app zippy_cash https://github.com/tanaygshah/cash_on_click.git
* bench install-app zippy_cash
